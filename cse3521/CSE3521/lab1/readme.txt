Lab1 is written in Java. There are three files, Node.java, AStar.java, 
IDDFS.java
Node.java contains the Node class. AStar.java contains the solution to
the A* approach. IDDFS contains the depth-limited DFS approach.
(Following commands shall be excuted in the project root folder where the 
src folder is in.)
Use the following command to compile:
javac -d ./ src/com/HYB/*.java
To run the depth limited DFS implementation, using the following command:
java com.HYB.IDDFS
To run the A* implementation, using the following command:
java com.HYB.AStar
